# 🚴 Bike Station Performance & Availability Dashboard

## 📌 Overview

This project presents an interactive Power BI dashboard developed to monitor and analyze bike station operations. The dashboard enables stakeholders to track bike availability, station capacity, utilization, and operational performance through dynamic visualizations and KPIs.

---

## 🎯 Objectives

- Monitor bike availability across stations.
- Analyze station capacity and utilization.
- Identify high-demand and underutilized stations.
- Support operational decision-making through interactive dashboards.

---

## 🛠️ Technologies Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel

---

## 📂 Dataset

The dataset contains information such as:

- Station Name
- Available Bikes
- Available Bike Stands
- Capacity
- Status
- Contract Name
- Latitude & Longitude
- Last Updated

---

## 🧹 Data Preparation

- Removed null and duplicate records
- Cleaned text using Trim & Clean
- Created Date Dimension
- Implemented Star Schema
- Created relationships between Fact & Dimension tables

---

## 📊 KPIs

- Total Available Bikes
- Total Available Bike Stands
- Total Capacity
- Utilization %
- Full Stations
- Empty Stations

---

## 📈 Dashboard Features

- KPI Cards
- Interactive Slicers
- Map Visualization
- Capacity Analysis
- Utilization Trend
- Status Distribution
- Regional Performance
